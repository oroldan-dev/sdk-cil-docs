sendCoordinatesToControl

fun sendCoordinatesToControl(latitude: Double, longitude: Double)

Sends geographic coordinates (latitude and longitude) to a control system after validating the input.

This function checks whether the provided latitude and longitude are valid, and if so:

  • Assigns the coordinates to global variables (locationData.latitude and locationData.longitude).

  • Calls handleOnFinished to signal that the coordinate setting process has completed successfully.

If the coordinates are invalid:

  • Logs an error message indicating the invalid range for latitude and longitude.

Note: The function includes commented-out code for formatting and sending the coordinates as byte arrays, but these steps are not currently in use.

Parameters

latitude

The latitude coordinate to be sent (must be between -90 and 90).

longitude

The longitude coordinate to be sent (must be between -180 and 180).
