getMeasurements

fun getMeasurements()

Retrieves and processes measurement data from the connected device. Configures notifications and listens for responses based on the provided sector value and data type. Processes the buffered data and send it over Data Listener. On success:

  • The handleData method is called to process the provided dataMap and dataType.

  • A log message indicating the success of the operation is generated with the dataType.

  • The clearBuffer method is called to clear any buffered data for the given dataType.

  • The handleOnFinished method is called with a success flag (true).

On failure: The handleError method is called to log the error.

  • The handleOnFinished method is called with a failure flag (false).

  • The clearBuffer method is called to clear any buffered data for the given dataType.
