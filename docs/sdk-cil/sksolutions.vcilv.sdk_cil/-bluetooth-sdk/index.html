BluetoothSdk

class BluetoothSdk

Bluetooth SDK class which allows you to search, connect, and extract data from your VCILV devices using fast and accurate operations.

This SDK provides methods to manage Bluetooth Low Energy (BLE) connections and perform operations like sending commands, receiving notifications, and extracting data from devices.
